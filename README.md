# 简历生成器

> [!TIP] 提示
>
> 该项目更适合“前端宝宝”使用。

## 介绍

使用 markdown 语法在线书写并导出简历。

## 特点

- 以 [markdown 语法](https://en.wikipedia.org/wiki/Markdown) 为主。
- 内置图标：集成 [Font Awesome(free)](https://fontawesome.com/)。
- 支持自定义样式：集成 [Tailwind CSS](https://tailwindcss.com/)。

## 快速上手

1. "编辑区" 修改内容。
2. `Ctrl + P` 打印简历。
